# Personal Portfolio Website

## Overview
This project is a personal portfolio website designed to showcase my skills, projects, and experience as a Software Developer. The website focuses on clean UI/UX, responsiveness, and modern visual aesthetics.

---

## Features
- Modern and visually appealing UI design
- Fully responsive layout (mobile, tablet, desktop)
- Smooth scrolling and section-based navigation
- Scroll reveal animations
- Interactive hover effects
- Contact form with validation
- Resume download functionality
- Clean and structured layout

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Git & GitHub
- GitHub Pages (Deployment)

---

## Key Functionalities
- Dynamic navbar behavior on scroll
- Active navigation link highlighting
- Intersection Observer-based animations
- Form interaction handling
- Responsive grid layouts

---

## Challenges Faced
- Designing a layout that maintains visual balance across screen sizes
- Implementing smooth animations without affecting performance
- Ensuring consistent spacing and typography
- Managing UI interactions and hover states
- Debugging responsiveness issues

---

## Learnings
- Improved understanding of responsive design principles
- Better control over UI/UX consistency
- Practical experience with animations & interactions
- Strengthened Git workflow and deployment practices
